# 🧬 Humanity Progression Game

A drag-and-drop sandbox game where players start as early nomadic humans and combine elements to evolve through history — from survival to advanced civilization.

---

## 🚀 Features

- 🖱️ Drag-and-drop element system  
- 🔗 Logical combination mechanics  
- 📈 Progression through human history  
- 📊 Discovery tracking system  
- 🧠 Experiment-based gameplay  

---

## 🧠 How It Works

1. Start with basic elements (Human, Fire, Water, etc.)
2. Drag one element onto another
3. If the combination is valid:
   - A new element is created
   - Both original elements are removed
4. Keep combining to unlock new discoveries

---

## 🧪 Example Combinations

| Element 1 | Element 2 | Result |
|----------|----------|--------|
| Human | Stone | Tool |
| Tool | Human | Hunting |
| Hunting | Animal | Meat |
| Fire | Meat | Cooked Food |
| Agriculture | Village | Civilization |
| Science | Technology | Future |
